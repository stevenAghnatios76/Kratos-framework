# CI Setup Checklist
- [ ] CI platform confirmed by user (not just auto-detected)
- [ ] Pipeline stages defined (build, lint, test, coverage)
- [ ] Quality gate thresholds set
- [ ] Secrets management documented (required secrets, environment separation)
- [ ] Deployment strategy defined (staging, production, rollback)
- [ ] Monitoring and notifications configured (failure alerts, status badge)
- [ ] Pipeline config generated
- [ ] Gates are enforced (blocking, not advisory)
